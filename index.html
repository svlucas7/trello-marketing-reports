Centro de Ajuda

🚀 Guia de Início Rápido

1
Configure o Período

Defina as datas de início e fim do seu relatório na aba "Importar Dados".

2
Exporte do Trello

No seu quadro Trello: Menu (⋯) → Mais → Exportar como JSON

3
Importe os Dados

Cole o conteúdo do arquivo JSON na área de texto e clique em "Gerar Relatório".

4
Analise e Exporte

Navegue pelas abas Dashboard e Relatórios, depois exporte no formato desejado.

🔧 Solução de Problemas

❌ "Erro ao processar JSON"

Solução: Verifique se copiou todo o conteúdo do arquivo JSON. O arquivo deve começar com "{" e terminar com "}".

📊 "Nenhuma tarefa encontrada"

Solução: Confirme se o período selecionado está correto e se existem cartões ativos no quadro do Trello.

👥 "Colaboradores não aparecem"

Solução: Certifique-se de que os cartões no Trello têm membros atribuídos.

❓ Perguntas Frequentes

Os meus dados são seguros?

Sim! Todo o processamento é feito localmente no seu navegador. Nenhum dado é enviado para servidores externos.

Posso usar com qualquer quadro do Trello?

Sim, a ferramenta funciona com qualquer quadro do Trello que você tenha acesso para exportar.

Como são calculados os atrasos?

O sistema considera a data de vencimento dos cartões comparada com a data atual para calcular os dias de atraso.
